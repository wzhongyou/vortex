# Vortex — 多路召回引擎

Vortex 是一个基于现代 C++17 开发的高性能、多策略召回引擎。

支持可插拔的索引后端——倒排索引用于关键词搜索，向量索引用于语义搜索——并将多路结果融合为统一排序列表。

## 规划特性

- **倒排索引** — 分词 → 倒排列表 → BM25 / TF-IDF 打分
- **向量索引** — HNSW 图结构，支持 L2 / 内积 / 余弦距离
- **混合召回** — 并发查询多个索引，通过 RRF 或加权融合合并结果
- **可插拔架构** — 实现 `Index` 接口即可接入自定义召回策略
- **零拷贝序列化** — 基于 FlatBuffers 的索引持久化
